# AtmoAI

Monorepo scaffold: frontend (React/Vite) and backend (Python placeholder).

Frontend
- Path: `frontend/`
- Start:
```bash
cd frontend
npm install
npm run dev
```

Backend (placeholder)
- Path: `backend/`
- Install:
```bash
python -m pip install -r backend/requirements.txt
```
- Start (development):
```bash
uvicorn backend.app.main:app --reload --port 8000
```

Docs
- Path: `docs/` (empty)
